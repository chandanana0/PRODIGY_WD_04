Wistful Tunes

"Wistful Tunes" is an online Album store project created using Python with advanced administrative features. It allows admins to manage records, process orders, and track customer payments. Customers can preview albums before purchase, and members receive discounts based on their membership level (gold, silver, or platinum). The system also enables membership record management. This project showcases my expertise in developing efficient e-commerce solutions.
